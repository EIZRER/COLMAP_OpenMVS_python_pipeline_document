\chapter{УДИРТГАЛ}

\section{Судалгааны үндэслэл, ач холбогдол}

3D реконструкц нь робот техник, виртуал (VR) болон AR, соёлын өвийн дижитал архив, автономи систем зэрэг олон салбарт үндсэн технологийн үүрэгтэй. 2D зурагнуудаас 3D загвар үүсгэх нь хүний оролцоогүйгээр 3D загварчлал хийх, 3D хувьчлахад чухал ач холбогдолтой.

Уламжлалт 3D сэргээн босголтын аргууд болох Structure-from-Motion (SfM) болон Multi-View Stereo (MVS) нь бодит геометрийг өндөр нарийвчлалтайгаар сэргээдэг боловч дараах сул талуудтай:

\begin{itemize}
    \item \textbf{Гэрэлтүүлгийн хувирлаас өндөр хамааралтай}
    \item \textbf{Объектын гадаргуугийн тусгал, хээ, өнгөнөөс шалтгаалан алдаа үүсдэг}
    \item \textbf{Олон зураг шаарддаг}
    \item \textbf{Тооцооллын хувьд хүнд}
\end{itemize}

Тиймээс энэхүү судалгааны ажлын гол зорилго нь COLMAP болон OpenMVS ашиглан 3D
сэргээн босголтын бүрэн автоматжуулсан SfM+MVS pipeline болон хэрэглэгчийн графикийн
интерфейс (GUI) боловсруулж, тогтмол нөхцөлд зураг авах орчин ашиглан реконструкцийн
чанарыг тоон болон харааны аргаар үнэлэх явдал юм. Боловсруулсан систем нь цаашид
машин сургалтын аргуудыг нэгтгэх боломжтой өргөтгөх чадвартай суурь болж өгнө.

\section{Судалгааны зорилго, зорилт}

\subsection{Зорилго}

Олон өнцгийн 2D зургаас 3D загвар үүсгэх бүрэн автоматжуулсан SfM+MVS pipeline болон хэрэглэгчийн графикийн интерфейс (GUI) боловсруулж, тогтвортой гэрэлтүүлэгтэй зураг авах орчинд реконструкцийн чанарыг үнэлэхэд оршино.

\subsection{Зорилтууд}

\begin{enumerate}
    \item \textbf{Өгөгдөл цуглуулах:} Тогтмол гэрэлтүүлэг бүхий хяналттай орчинд объектыг олон өнцгөөс системтэйгээр зураг авах
    \item \textbf{Уламжлалт аргуудыг хэрэгжүүлэх:} COLMAP ашиглан SfM ба MVS реконструкц хийх
    \item \textbf{Хэрэглэгчийн интерфейс боловсруулах:} Бүх үе шатыг нэг цонхноос удирдах боломжтой PyQt5-д суурилсан GUI систем бүтээх
    \item \textbf{Автоматжуулсан pipeline бүтээх:} Зураг оруулахад SfM → MVS → Mesh generation алхмуудыг нэг товчны даралтаар ажиллуулдаг, 9 үе шаттай автоматжуулсан pipeline боловсруулах
    \item \textbf{Гүйцэтгэлийг үнэлэх:} Chamfer Distance болон RMSE (Root Mean Square Error) хэмжүүрийг ашиглан реконструкцийн чанарыг тоон аргаар хэмжих
    \item \textbf{Matching параметрүүдийн нарийвчлалд үзүүлэх нөлөөг туршилтаар тодорхойлох:} Matcher-ийн сонголт, хослолын стратеги болон overlap параметрийн реконструкцийн чанарт үзүүлэх нөлөөг гурван туршилтын цуваагаар үнэлэх
\end{enumerate}

\section{Судалгааны хамрах хүрээ}

Судалгааны ажлын хүрээ дараах байдлаар хязгаарлагдана:

\begin{itemize}
    \item \textbf{Объект:} Жижиг хэмжээтэй, нэг ширхэг хатуу биет объект
    \item \textbf{Зураг авах орчин:} Тогтмол гэрэлтүүлэг бүхий хаалттай камерын төхөөрөмж
    \item \textbf{Арга:} SfM + MVS аргуудыг ашиглан автоматжуулсан 3D реконструкц
    \item \textbf{Програм хангамж:} COLMAP, OpenMVS, Python, PyQt5
    \item \textbf{Гаралт:} Wavefront OBJ форматтай 3D загвар
\end{itemize}

\section{Судалгааны шинэлэг тал}

Энэхүү судалгааны ажил дараах өвөрмөц онцлог, шинэлэг талтай:

\begin{enumerate}
    \item \textbf{Бүрэн автоматжуулсан 9 үе шаттай pipeline} — зураг оруулахад COLMAP болон OpenMVS-ийн бүх үе шатыг дарааллаар автоматаар гүйцэтгэдэг, нэг товчны даралтаар ажиллах pipeline боловсруулсан
    \item \textbf{PyQt5 GUI бүхий автоматжуулсан систем} — зураг авалтаас mesh гаралт хүртэл бүх 9 алхмыг нэг цонхноос удирдах боломжтой хэрэглэгчийн интерфейс
    \item \textbf{Автомат pipeline} — зураг оруулахад SfM → MVS → Mesh → Texture хүртэлх бүх үе шатыг дарааллаар автоматаар гүйцэтгэдэг систем
    \item \textbf{Mesh чанарын тоон ба дүрслэх үнэлгээ} — зөвхөн харааны үнэлгээ бус Chamfer distance болон RMSE ашигласан
    \item \textbf{Бодит хэрэглээнд шууд ашиглаж болох шийдэл} — жижиг лабораторид, музей, соёлын өвийн сканнерт ашиглаж болох прототип
\end{enumerate}
